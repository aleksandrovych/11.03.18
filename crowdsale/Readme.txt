  Этот проект содержит тестовую транзакцию которая отправляет ефир на кроусейл
и получает определенное количество токенов.
  Токены майнятся автоматически при создании контракта токена, их заданное количество (1 000 000 000).
Трафл разлочивает все аккаунты( в 2_internal_migrations.js нужно добавить свои пароли в массив в этом файле).
  Деплоится краудсейл с первого аккаунта. Затем ищет акк на котором есть минимум 0.001 ефира
и посылает его на краудсейл, это и есть тестовая транзакция. После этого проверяет, что баланс токенов
для отправителя эфира пополнился.
  Тестировал на rinkeby, деплоить - truffle migrate --reset из папки проекта
Требования - на первом акке больше 1-го ефира. еще один акк на котором не меньше 0.001 ефира.
